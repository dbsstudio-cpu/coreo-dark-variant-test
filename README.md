# COREO Dark Variant v0.1

本資料夾是 COREO 的深色視覺實驗版，從主線 v2.7 複製而來，只用於手機 A/B 測試。

## 測試目的

- 測試黑鈦展示艙風格是否比明亮版更有遊戲感。
- 測試冷光管線在深色背景下是否更容易看出連續路徑。
- 測試主角冰藍、反派紅光、LIGHT CORE 分級在暗色場景中的辨識度。

## 邊界

- 不取代主線 `APP/`。
- 不新增追逐 AI。
- 不改碰撞邏輯。
- 不改部署主線網址。
